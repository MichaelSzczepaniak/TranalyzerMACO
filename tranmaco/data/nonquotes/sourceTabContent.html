Branch Descriptions

There are two branches on this project. The master is at https://github.com/MichaelSzczepaniak/TranalyzerMACO/tree/master and the demo is at https://github.com/MichaelSzczepaniak/TranalyzerMACO/tree/shinydemo. These branches are described below.

The demo Branch

This version of the app is running at: https://michael-szczepaniak.shinyapps.io/tranmaco/

The distinguishing feature of the demo version is the select list in the Company field. Data for 10 stocks over a 10 year period were loaded with the application files. This was done to improve user experience by lowering the risk of a user having trouble obtaining data from a live/real-time quote query.

The master Branch

This verion of the app allows the user to query end-of-day quote data from Yahoo. This will eventually be the only difference between master and demo branches once changes from demo have been merged back into master.

This app was developed as a protype to see how some simple ideas might be implemented. A more feature-rich version capable of more advanced simulations is planned to be done in Python.
