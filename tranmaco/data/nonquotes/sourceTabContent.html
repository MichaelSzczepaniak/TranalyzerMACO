Branch Descriptions

The master living at https://github.com/MichaelSzczepaniak/TranalyzerMACO is the version that is deployed and running at: https://michael-szczepaniak.shinyapps.io/tranmaco/

Other branches typically start with the prefix dev_ for development followed by a descriptor such as cv for cross-validation. The development branches are typically pulled from the master.

This app was developed as a protype to see how some simple ideas might be implemented. A more feature-rich version capable of more advanced simulations is planned to be done in Python.
